# 🍕 Pizza Ordering MERN Application

 Objective

The objective of this project is to develop a full-stack pizza ordering web application using the MERN stack (MongoDB, Express.js, React.js, and Node.js). The application allows users to customize pizzas, place orders, and enables admin management of orders and inventory.

---

 Features

 User Module

* User Registration & Login
* Authentication System
* Pizza Dashboard
* Pizza Customization
* Place Orders
* Order Tracking

 Admin Module

* Admin Dashboard
* View All Orders
* Update Order Status
* Inventory Management

---

 Technologies Used

 Frontend

* React.js
* React Router DOM
* Axios

 Backend

* Node.js
* Express.js

 Database

* MongoDB

 Other Tools

* JWT Authentication
* Socket.io



 Project Structure

* Frontend (React)
* Backend (Node.js + Express)
* MongoDB Database



 Steps Performed

1. Setup MERN Stack Environment
2. Created Authentication System
3. Built Pizza Dashboard UI
4. Developed Pizza Builder
5. Connected Frontend with Backend APIs
6. Stored Orders in MongoDB
7. Created Admin Dashboard
8. Added Inventory Management
9. Implemented Order Status Updates

 Outcome

Successfully developed a responsive full-stack pizza ordering application with user authentication, pizza customization, order management, and admin functionalities.
